Implementação de um código em linguagem C que utiliza a força bruta para identificar se dois grafos são isomorfos. 
Ao final, caso os grafos sejam isomorfos, apresenta a função de isomorfismo.

Observações:
    - Os grafos verificados deverão ser lidos de dois arquivos com o mesmo formato utilizado no trabalho anterior;
    - Devem ser utilizadas estratégias que melhorem a eficiência da força bruta, tais como verificação de propriedades equivalentes e classificação de vértices pelo grau;
